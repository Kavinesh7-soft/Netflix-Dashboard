# 🎬 Netflix Content Analysis Dashboard

## 📌 Overview

This project is an interactive Netflix Content Analysis Dashboard developed using Microsoft Excel. It provides valuable insights into Netflix's content library through interactive charts, Pivot Tables, and slicers.

---

## 🎯 Objective

To analyze Netflix content and identify trends based on:

- Movies vs TV Shows
- Country
- Genre
- Rating
- Release Year

## 📈 Dashboard Features

- Movies vs TV Shows Distribution
- Top 10 Countries Producing Netflix Content
- Year-wise Growth
- Most Popular Genres
- Rating Distribution
- Interactive Filters

## 🛠 Tools Used

- Microsoft Excel

- Pivot Charts
- Slicers
- Data Cleaning

## 📂 Dataset Columns

- Show ID
- Title
- Type
- Country
- Genre
- Rating
- Release Year
- Duration

---

## 📌 Key Insights

✔ Movies dominate Netflix's content.

✔ USA has the highest number of titles.

✔ Content increased rapidly after 2015.

✔ TV-MA is the most common rating.

✔ Documentaries are among the most popular genres.
